## 条件控制-if~else语句(重点)

```
格式:
if 条件语句:
	代码块1
else:
	代码块2

a=10
b=20

if "sss":#0表示假,非0表示真,空字符串为假,非空字符串为真
    print("a<b")
else:
    print("a>=b")


注意：
1）Python没有像其它大多数语言一样使用“{}”表示语句体，
而是通过语句缩进来判断语句体，缩进默认为4个空格。
2）可以使用比较、逻辑表示条件语句
3）可以使用in和not in表示包含关系
4）甚至可以使用布尔类型的判断

案例:从控制台输入一个数，判断这个数是否是偶数
n=int(input("请输入你的数字:"))
if n%2:
    print("不是偶数")
else:
    print("是偶数")
    
#案例:从终端输入用户和密码,如果都正确,打印欢迎登入,如果有一个不正确就打印用户名或密码不正确
#用户名在str1='makermaker2maker3maker4',密码在str2='0001000200030004'


```

## if-elif-else(重点)

```
格式:
if 条件语句1:
	代码块1
elif 条件语句2:
	代码块2
elif 条件语句3:
	代码块3
else:
	代码块4

注意：
1、每个条件后面要使用冒号（:），表示接下来是满足条件后要执行的语句块
2、使用缩进来划分语句块，相同缩进数的语句在一起组成一个语句块
3、在Python中没有switch – case语句

案例:根据控制台输入的成绩，输出对应的等级
	90以上：优秀
	80~90：良好
	60~80：还行
	60以下：加油吧，少年

#案例:从终端输入一个字符串,判断是否为纯数字,是否为纯大写字母,是否为纯小写字母,然后打印对应的字符串

```

## if-elif-else的嵌套

```
格式:

	if 条件4:
		代码块1
	else:
		代码块4
	
elif 条件2:
	if 条件5:
		代码块2
	else:
		代码块5
else:
	代码块3


注意:为了代码的维护,建议嵌套不要超过3层
案例:用户身份验证: 用户名输入admin, 密码输入123456, 验证成功,用嵌套的方法
yzm=3344
user="admin"
password="123456"

if yzm==3344:
    #如果进入到这,表示你输入的验证码正确,然后我才会判断用户名和密码
    if user=="admin" and password=="123456":
        print("欢迎登录")
    else:
        print("输入的用户名或密码错误")
else:
    print("验证码错误")
    
#案例,猜数字,有一个数字,389,猜中第一位才可以继续往下猜,没猜中就现在'很遗憾,没猜中',猜中了就现在
#'恭喜你,猜中了,请继续',全猜中,打印'太棒了,全中'


```

## range()函数(重点)

```
range()函数
如果你需要遍历数字序列，可以使用内置range()函数。它会生成数列
你也可以使用range指定区间的值
也可以使range以指定数字开始并指定不同的增量(甚至可以是负数，有时这也叫做'步长')
格式:
for i in range(开始[,结束,步长]):
	print(i)

#打印1-100,生成的数列是1-100,不包含101
for i in range(1,101):
    print(i)
    
#这里的10表示结束,生成的数列的0-9
for i in range(10):
    print(i)

#生成1,3,5,7,9的数列
for i in range(1,10,2):
    print(i)
    
#生成10,8,6,4的数列
for i in range(10,2,-2):
    print(i)
    
#案例:用for加range()函数输出10-20的偶数,反向打印
for i in range(20,9,-2):
    print(i)
    
#案例:打印以下图形
0123456789
101112131415
1617181920

```



## break,continue

```
break:退出循环,嵌套的循环中一个break只能跳出一个循环
while 1:
    print("外循环")
    while 1:
        print("内循环")
        break#只能跳出一个循环


continue:退出本次循环
for i in range(11):
    if i==6:
        continue#这个语句后面的代码就不执行,回到循环开头
    print(i)
```

## 列表(重点)

```
一个变量每次只能存储一个数据，如果一次性存储多个数据，怎么做?答:用列表
1.列表是一种数据类型,也是一个容器,里面可以装数据
2.列表的创建:
格式一:
列表名=[数据1,数据2,数据3,.....]
比如:mylist=[1,2,3,4,5,6] #在创建列表时就定义好了内存空间的大小,后期通过下标不能越位操作
格式二:
列表名=[]  #这种定义方式,定义的列表是没有空间的,不能通过下标来存储数据
比如:mylist2=[]
3.列表的数据存储或取出
mylist[0]=100
print(mylist)#[100,2,3,4,5,6]
mylist[6]=200
print(mylist)#报错
mylist2[0]=100#报错

注意，
1、list写在方括号之间，元素用逗号隔开
2、和字符串一样，list可以被索引和切片
#切片
print(mylist[0:3])
3、list可以使用+操作符进行拼接
mylist3=['a','b',22]
print(mylist+mylist3)
4、list中的元素是可以改变的
5.列表中的元素可以是不同的数据类型
```


## 列表的操作-添加序列extend(重点)

```
extend(seq)   把一个序列seq的内容添加到列表中
mylist=[1,2]
mylist.extend("hello")
print(mylist)#[1, 2, 'h', 'e', 'l', 'l', 'o']
mylist.extend([10,20,30,40])
print(mylist)#[1, 2, 'h', 'e', 'l', 'l', 'o', 10, 20, 30, 40]

#拆解字符串
mystr="world"
mylist2=[]
mylist2.extend(mystr)
print(mylist2)#['w', 'o', 'r', 'l', 'd']

#案例:从终端中输入5个数字,以空格分开,然后把这些数字组成一个列表,然后加入到mylist2=[1,2,3,4,5]列表中
#注意,要把列表中字符数字变为数字
n=input("请输入5位的数字:")
mylist=n.split()
k=0
for i in mylist:
    mylist[k]=int(i)
    k+=1
print(mylist)
mylist2=[1,2,3,4,5]
mylist2.extend(mylist)
print(mylist2)
```
